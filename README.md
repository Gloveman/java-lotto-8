# java-lotto-precourse
## **프로젝트 소개**
### 로또 발매기 구현
사용자가 입력한 금액만큼 로또를 발행하고, 각각의 당첨 내역 및 수익률을 알려주는 프로그램

## **주요 기능**
- 주어진 금액으로 로또 구매
- 당첨번호 + 보너스 번호 입력
- 결과 확인

## **구현할 기능**
- [x] 6개의 숫자를 가진 로또 발행
- [x] 일치 개수로 등수 판별, 당첨금 계산
- [x] 주어진 당첨 번호 + 보너스 번호로 일치 개수 구하기
- [x] 주어진 금액만큼의 로또 발행
- [x] 금액 예외 처리 - 1000원으로 나누어 떨어지지 않는 경우
- [x] 로또 발행 예외 처리 - 중복된 숫자가 있는 경우
- [x] 로또 발행 예외 처리 - 숫자가 범위를 벗어난 경우
- [x] 구매 금액, 당첨 번호 + 보너스 번호 사용자 입력
- [x] 잘못된 입력 처리 - 아무것도 입력을 하지 않은 경우
- [x] 잘못된 입력 처리 - 숫자가 아닌 다른 문자열을 입력한 경우
- [x] 잘못된 입력 처리 - 숫자가 범위를 벗아난 경우
- [x] 잘못된 입력 처리 - 당첨 번호와 보너스 번호 중 중복된 숫자가 있는 경우
- [ ] 당첨 내역, 수익률 출력
